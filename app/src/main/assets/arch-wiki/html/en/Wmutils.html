Wmutils

Using wmutils (window manipulation utilities), you can do a multitude of simple tasks such as listing windows, moving/resizing/teleporting them, changing their visibilty, changing their stacking order, and more. You can even chain commands together to complete actions that other window managers cannot do.

Installation

Install the wmutils-gitAUR package for the core functionality.

wmutils, however, is only a windows manipulation utility. A window manager and a keybinding controller (sxhkd, xbindkeys and others) are also required. In the following sections, swm-gitAUR and sxhkd will be used.
