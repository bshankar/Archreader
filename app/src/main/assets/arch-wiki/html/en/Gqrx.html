gqrx is a software defined radio receiver powered by the GNU Radio SDR framework and the Qt graphical toolkit.

Installation

Install gqrx or gqrx-gitAUR.

Usage

In order to use gqrx with PulseAudio, run: 

$ gqrx

If you want to use gqrx with alsa, run: 

$ gqrx-alsa
