Vivaldi

Vivaldi is a new web browser from former Opera team members, based on Chromium and focused on personalization aspects.

Installation

Vivaldi can be installed with vivaldiAUR or vivaldi-betaAUR or vivaldi-snapshotAUR. Alternatively you can add herecura's repo, where you can find prebuilt packages.

Media playback

For additional support concerning audio and video playback, install corresponding package for your version of choice: vivaldi-ffmpeg-codecsAUR, vivaldi-beta-ffmpeg-codecsAUR or vivaldi-snapshot-ffmpeg-codecsAUR.

You may install pepper-flashAUR to play flash media

Troubleshooting

Unable to set Vivaldi as default web browser

You have to edit default applications manually. Vivaldi is not able to recognize, how to do it by itself.
