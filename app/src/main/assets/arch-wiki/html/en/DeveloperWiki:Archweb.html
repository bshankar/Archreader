DeveloperWiki:Archweb

Arch Linux web development is made using Python with the Django framework. Development and commit rights are restricted to Arch Linux developers.

If you are not a developer, you are nevertheless free to contribute:

  • clone the Git repository using https://projects.archlinux.org/archweb.git/ or the mirror git://github.com/archlinux/archweb.git;
  • read the project readme to learn the installation steps and requirements;
  • make a patch;
  • and ping some Arch Linux developers either by mail or IRC to see your patch committed in your name.
