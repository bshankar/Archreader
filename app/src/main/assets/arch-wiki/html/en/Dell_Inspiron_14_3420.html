Hardware

Audio

There are two options to get audio working: ALSA and PulseAudio. With ALSA the sound works well, both headphone jacks work and volume can be set independently. With Pulse Audio you will generally get better quality and louder sound.

Video

The notebook comes with the Intel HD 4000 GPU, which uses the xf86-video-intel driver. See: Intel for details.

Keyboard

Keyboard worked out of the box.

Touchpad

Touchpad worked out of the box. To enable scroll and more, install xf86-input-synaptics. With the default configuration file it will work well, but if you want to fine tune the behavior.

See: Touchpad Synaptics.

Wireless

This laptop comes Broadcom Corporation BCM43142 wireless card, which requires broadcom-wl-dkmsAUR

See: Wireless network configuration.

Toggle between HDMI/Monitor

See xrandr.

USB, SD card slot, ethernet, VGA, HDMI, webcam

All work out of the box.
