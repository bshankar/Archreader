Specs

  • Intel Xeon e3-1270v3
  • 32GB ECC Ram
  • 2x240GB SSD as Raid1
  • 30TB Traffic
  • 1000Mbit/s Uplink

Partition layout

/dev/sda1 and /dev/sdb1 all the space -> /dev/md0

/dev/md0 contains an LVM (vg0) which contains the following volumes:

  • home /home
  • root /
  • srv /srv
  • log /var/log
  • mysql /var/lib/mysql
