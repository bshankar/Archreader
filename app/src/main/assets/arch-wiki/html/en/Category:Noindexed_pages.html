Category:Noindexed pages

This is a hidden category, i.e. it is not displayed at the bottom of member pages among normal categories.

Users can choose to see hidden categories in a separate "Hidden categories" list by checking "Show hidden categories" in Special:Preferences#Appearance.

This maintenance category tracks pages that have the __NOINDEX__ behaviour switch added to them, so that they do not show up in web search engines. This category is added automatically by the MediaWiki software. The category name is defined in MediaWiki:Noindex-category (see Commons:MediaWiki:Noindex-category for a switch based on namespace).

Only pages in the User and User_talk namespaces can belong here; all others should have __NOINDEX__ removed.
