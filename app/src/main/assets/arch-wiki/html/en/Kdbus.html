kdbus

This article is being considered for archiving.

Reason: There have been no kdbus releases for kernels 4.7 and up, and the project seems abandoned as developers are switching attention to BUS1 for the future of in-kernel IPC mechanism (Discuss in Talk:Kdbus#)

kdbus is an alternative in-kernel implementation of D-Bus.

Installation

Install the kdbusAUR package, then add kdbus=1 to the kernel parameters.

After rebooting, you can verify a working kdbus installation with: 

$ dmesg | grep kdbus
kernel: kdbus: initialized

Also check that /sys/fs/kdbus/ contains the 0-system/ folder, denoting the system bus. Another sign of a working kdbus installation is that, after invoking busctl, the DESCRIPTION column should not be entirely empty for all services.
