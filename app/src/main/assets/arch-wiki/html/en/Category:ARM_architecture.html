Category:ARM architecture

Note:
  • Support for the ARM architecture is provided by Arch Linux ARM, which is a different project than Arch Linux. Seek support on the Arch Linux ARM forums.
  • The ARM abbreviation also refers to Arch Rollback Machine, which however provides packages for the Arch Linux official architectures i686 and x86_64.

Arch Linux ARM is a port of Arch Linux to the ARM architecture. Pages in this category provide some additional tips and tricks for various ARM devices. They are not intended as a replacement for the official documentation, but were contributed before the Arch Linux ARM project restarted its own ALARM wiki.

For new articles, please consider first whether you can help the Arch Linux ARM project's wiki to a good start with your ARM specific contributions. Please also consider a duplication of topics between both wikis is counter-productive for all involved. If you create an article for a new topic on the ALARM wiki, you are welcome to create a short article redirecting the reader to it in this category.
