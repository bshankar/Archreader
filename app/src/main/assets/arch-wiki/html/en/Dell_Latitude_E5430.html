Dell Latitude E5430

The Dell Latitude E5430 is a business line laptop made for corporate users who have a need for durability. This article will tell you how to get the basic components of the laptop running with Arch.

Installation

You can follow the Installation guide to get yourself up and running. Just few exceptions mentioned below.

Wireless

Broadcom BCM5761

This card requires the proprietary broadcom-wlAUR driver, available in AUR (alternatively, you can use its dkms version, broadcom-wl-dkmsAUR).

Graphic Interface Configuration

Issue with XKeyboardConfig

While starting the graphic interface, you might come across with the bug #77504 reported in freedesktop bugzilla. It seems to apply to both Xorg and Xwayland. During the display server startup, the XKEYBOARD keymap compiler (xkbcomb) reports: 

> Warning:          Compat map for group 2 redefined
>                   Using new definition
> Warning:          Compat map for group 3 redefined
>                   Using new definition
> Warning:          Compat map for group 4 redefined
>                   Using new definition
Errors from xkbcomp are not fatal to the X server

I (josephgbr) followed the solution provided in the forum topic Xkeyboard/xkbcomp gives warnings and it seems to work fine: Comment out the following lines in the file /usr/share/X11/xkb/compat/basic. The double forward-slash is the comment symbol. 

 //    group 2 = AltGr;                                                                
 //    group 3 = AltGr;
 //    group 4 = AltGr;

General Configurations

PC Speaker

The E5430 has an awfully loud PC speaker. To disable it, ensure the module pcspkr is not loaded during bootup: 

/etc/modprobe.d/nobeep.conf
#Do not load the 'pcspkr' module on boot.

blacklist pcspkr

You should follow instructions for installing cpufrequtils and pm-utils. There have been no problems with suspending and hibernating this laptop - just ensure you have enough swap.

Synaptics Touchpad

Make sure you install the package xf86-input-synaptics as it is required by the touchpad
