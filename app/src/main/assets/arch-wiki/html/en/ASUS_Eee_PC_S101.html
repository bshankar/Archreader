Installation

To boot from a USB stick, disable boot booster in BIOS Setup.

Xorg

Install the xf86-video-intel package.

Touchpad

See Touchpad Synaptics.

Camera

Supported by stock kernel, see Webcam Software

Wireless

  • AR9281/PCIe Full MiniCard 91, Single band 1x2 configuration

Hot keys

Hotkey Action Notes
Fn+F1 Sleep
Fn+F2 Wifi Atheros Communications Inc. AR928X Wireless Network Adapter (uses the ath9k module)
Fn+F5/F6 Brightness BIOS v1504 has controlled it.
Fn+F10/F11/F21 Volume Mute/Lower/Raise Setup required, see Acpid
