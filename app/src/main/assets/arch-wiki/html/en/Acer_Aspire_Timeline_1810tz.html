Fan control

Acerhdf [1] is a module you can load to control the fan-speed. Download the latest version, unpack it and follow instructions in the readme file. You probably need to add your bios-version in the config-file. Your bios-version is in the error message when you load the module. Add acerhdf in your rc.conf. If it doesn't work, you can also follow these instructions: Acer Aspire One#acerhdf

Troubleshooting

Touchpad

If the touchpad becomes unresponsive, add the following: [2] 

~/.xinitrc
xinput --set-prop "SynPS/2 Synaptics TouchPad" "Synaptics Two-Finger Pressure" 280
