Create an SSH Key

You will need an ssh key to access git on the Arch Women servers. If you don't have a key, you can create one by running: 

 ssh-keygen

For more information on creating key pairs see, SSH keys

Share Your Public Key

After generating key pairs you should have a public and private key in ~/.ssh. The public key has the extension .pub. To paste your public key to ptpb.pw do: 

cat ~/.ssh/id_rsa.pub | curl -F c=@- https://ptpb.pw/

Make sure the key you paste has the .pub extension, you don't want to publicly share your private key.
