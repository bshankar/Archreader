Jruby

Installation

Install the jruby package.

Note: jruby will require either openjdk or jre in order to run
  1. Download the latest Jruby from http://jruby.org/download to '/opt/jruby'
  2. Add 'opt/jruby/bin' to your path
# export PATH=$PATH:/opt/jruby/bin

Testing

  • Run the following command to test jruby
# jruby -v
