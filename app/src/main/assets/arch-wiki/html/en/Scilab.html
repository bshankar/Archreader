Scilab is free and open-source (licensed under the GPL-compatible CeCILL license), cross-platform (compatible with Linux, macOS and Windows NT) high-level interpreted programming language primarily intended for technical computing. Scilab is also the name of an integrated development environment for the Scilab programming language. Syntactically, the Scilab programming language, is similar to MATLAB, although not as similar as GNU Octave. Scilab is implemented in C, C++, Fortran and Java. Scilab is developed by Scilab Enterprises.

Installation

Scilab is not in the official repositories of Arch Linux, so it must be installed either from unofficial user repositories, such as pkgbuild-current, or from the Arch User Repository (AUR).
